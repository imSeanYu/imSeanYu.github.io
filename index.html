PROJECTS



    Crossfade

    Crossfade is a single-page music sharing application with seamless song playback inspired by SoundCloud.
    It is engineered with React.js front-end and Redux RESTful API, with a Rails back-end.
    Users may create an account and upload song/image files which is stored and fetched to and from AWS S3 servers through the use of Paperclip gem and AJAX requests.
    The app is currently hosted on Heroku.

    LIVE | REPO

    TypeDefenseForce

    TypeDefenseForce is a typing game inspired by z-type.
    The app is written mostly in vanilla JavaScript with the help of jQuery library, and is rendered in HTML5 Canvas.
    Try to set a high score and leave your mark in the "Hall of Fame," which is built through an integration of Google's Firebase Real-time Database.

    LIVE | REPO

    DOMToretto

    DOMToretto is a lightweight DOM manipulation library inspired by jQuery. It features an easy-to-use API which lets users traverse and manipulate DOM's. The live demo is the classic snake game which utilizes DOMToretto to manipulate DOM's and render the page.

    LIVE | REPO
